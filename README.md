# Latihan Fullstack CRUD App

Aplikasi web sederhana dengan fitur autentikasi dan CRUD menggunakan React dan Express.

## Fitur
- Register dan Login dengan JWT
- Tambah, edit, hapus, dan tampil data item
- Proteksi halaman dashboard dengan token
- Semua aksi CRUD dilakukan dalam modal
- UI rapi menggunakan React-Bootstrap

## Struktur Folder
- `latihan-frontend/` → React + React Router
- `latihan-backend/` → Express + JWT + file JSON

## Cara Menjalankan

### Backend
```bash
cd latihan-backend
npm install
node index.js

### Front end
cd latihan-frontend
npm install
npm start
